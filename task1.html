HTML and CSS

HTML və CSS internet səhifələrinin yaradılmasında əsas rol oynayan iki dildir. HTML (Hyper Text Markup Language) internet səhifələrinin mətn, şəkil və linklərinin strukturu və tənzimlənməsi üçün istifadə olunur. CSS (Cascading Style Sheets) isə səhifənin görünüşünü tənzimləmək üçün istifadə olunur.

Javascript

JavaScript vasitəsilə, internet səhifələri həm dinamik, həm də interaktiv olur. Özəlliklərindən biri də, istifadəsi çox asandır və müxtəlif tərəflər tərəfindən geniş istifadə edilir.

Python

Python proqramlaşdırma dilləri arasında ən populyar dillərdən biridir və müxtəlif məqsədlərlə, həm də istifadəçi təcrübəsi səviyyəsinə görə, seçilir.
